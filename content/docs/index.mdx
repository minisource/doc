# Welcome to MiniSource

MiniSource is a comprehensive, production-ready microservices platform built with Go, designed to accelerate application development with enterprise-grade features out of the box.

## What is MiniSource?

MiniSource provides a complete suite of microservices that handle common application needs:

- **Authentication & Authorization** - JWT-based auth with role management and service-to-service authentication
- **Notifications** - Multi-channel notifications (SMS, Email, Push, In-App) with template management
- **Storage** - File storage and management with multiple backend support
- **Payment** - Payment gateway integration and transaction management
- **Scheduler** - Distributed job scheduling and cron management
- **API Gateway** - Centralized API gateway with rate limiting and routing

## Architecture

MiniSource follows microservices architecture principles with:

- **Go-based Services** - High-performance, compiled binaries
- **gRPC Communication** - Efficient inter-service communication
- **PostgreSQL** - Robust data persistence
- **Redis** - Caching and session management
- **Docker** - Containerized deployment
- **OpenTelemetry** - Distributed tracing and observability

- Health checks and metrics
- Graceful shutdown

### Developer-Friendly
- Comprehensive documentation
- SDKs for all services

- Horizontal scaling support

- Database migrations
- Distributed tracing
- Multi-tenancy support

## Technology Stack

- **Language**: Go 1.24+
- **Framework**: Fiber (HTTP), gRPC
- **Database**: PostgreSQL with GORM
- **Cache**: Redis
- **Logging**: Zap, Zerolog
- **Tracing**: OpenTelemetry
- **API Docs**: Swagger/OpenAPI

## Project Structure

```
minisource/
├── auth/           # Authentication service
├── gateway/        # API gateway
├── notifier/       # Notification service
├── storage/        # File storage service
├── scheduler/      # Job scheduler service
├── log/            # Centralized logging service
├── comment/        # Commenting service
├── ticket/         # Support ticket service
├── feedback/       # User feedback service
├── payment/        # Payment service (.NET)
├── go-common/      # Shared Go libraries
├── go-sdk/         # Client SDKs for all services
├── csharp-common/  # Shared .NET libraries
├── csharp-sdk/     # .NET client SDKs
├── go-template/    # Service template
├── infra/          # Infrastructure configs
└── doc/            # Documentation
```
